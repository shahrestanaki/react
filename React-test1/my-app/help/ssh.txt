---------- make local repository for yum
1- Install createrepo utility:
	* sudo yum install createrepo
2- Create a repository directory
	* mkdir /opt/rpms
3- Put RPM files into the repository directory
4- Create the repository metadata
	* createrepo <path_to_your_directory_with_rpms>
	for update: * createrepo --update <path_to_your_directory_with_rpms>
5- located in /etc/yum.repos.d/ directory
6- add  .repo extension file
	* local.repo:
	
		[nginx-stable]
		name=nginx stable repo
		baseurl=file:///etc/nginx.org/packages/centos/$releasever/$basearch/
		gpgcheck=1
		enabled=1
		gpgkey=file:///etc/nginx.org/keys/nginx_signing.key
		
		
module_hotfixes=true
7- check yum repo
	* sudo yum repolist:
		must seen local.repo
